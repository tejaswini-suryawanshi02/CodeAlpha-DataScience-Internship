# CodeAlpha Iris Flower Classification

# Objective

To classify Iris flowers into different species using Machine Learning.

# Dataset

Iris Dataset (CSV Format)

# Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn

# Algorithm

Random Forest Classifier

# Steps Performed

1. Loaded Iris dataset from CSV file.
2. Performed data preprocessing.
3. Encoded target labels.
4. Split dataset into training and testing sets.
5. Trained Random Forest model.
6. Evaluated model accuracy.
7. Visualized the dataset using Matplotlib.

# Result

The model successfully classified Iris flowers with high accuracy.

# Conclusion

Machine Learning algorithms can accurately classify Iris flower species based on flower measurements.

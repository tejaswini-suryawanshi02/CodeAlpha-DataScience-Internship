# 🚗 Car Price Prediction Dashboard

## 📌 Project Overview

This project was developed as **Task 3** for the **CodeAlpha Data Science Internship**.

The Car Price Prediction Dashboard is a Machine Learning application that predicts the selling price of used cars based on features such as manufacturing year, present price, kilometers driven, fuel type, seller type, transmission, and ownership.

The project is built using **Python**, **Streamlit**, and **Random Forest Regressor** to provide an interactive dashboard with data analysis and price prediction.

---

## 🎯 Objectives

- Analyze the car dataset.
- Visualize important insights using charts.
- Predict the selling price of a used car.
- Display model performance using evaluation metrics.

---

## 📂 Dataset Information

- Dataset Name: Car Data
- Total Records: **301**
- Total Features: **10**

### Features Used

- Year
- Present_Price
- Driven_kms
- Owner
- Fuel_Type
- Selling_type
- Transmission
- Car_Age

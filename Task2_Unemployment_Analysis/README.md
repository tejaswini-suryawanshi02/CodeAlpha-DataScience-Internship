# CodeAlpha - Unemployment Analysis with Python

## Project Overview
This project analyzes unemployment trends in India using Python and Streamlit. The dashboard provides interactive visualizations and insights based on unemployment data.

## Features
- Interactive Dashboard
- State-wise Analysis
- Covid-19 Impact Analysis
- Correlation Heatmap
- Employment vs Unemployment Visualization
- Key Insights and Recommendations

## Technologies Used
- Python
- Pandas
- Streamlit
- Plotly
- Seaborn
- Matplotlib

## Dataset
Unemployment in India Dataset (CSV)

## How to Run
python -m streamlit run unemployment_dashboard.py
